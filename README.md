# Develop ASP.NET Core SignalR application with React

To run an application;

```
cd DotnetIstanbulSignalRMeetup\ClientApp
npm install
cd DotnetIstanbulSignalRMeetup
dotnet run
```

Notes:
- You need to create a Twitter Application from https://apps.twitter.com/
    - Set *TWITTER_CONSUMER_KEY* environment variable
    - Set *TWITTER_CONSUMER_SECRET* environment variable
    - Set *TWITTER_ACCESS_TOKEN* environment variable
    - Set *TWITTER_ACCESS_TOKEN_SECRET* environment variable
